# EcoLabs_-002

This is a small demo farmer dashboard for EcoLabs.

Build (Tailwind CSS)

This project uses Tailwind CSS. To build the CSS for production locally:

1. Install dependencies:

	npm install

2. Build the CSS:

	npm run build:css

3. Open `index.html` in a browser or serve the folder with a static server.
